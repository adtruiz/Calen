# Willow & Co - Dog Park Management System

## Project Overview
Complete dog park management platform for bookings, payments, and admin control. Mobile app for customers, admin dashboard for park operators.

## Current Status: ACTIVE - Waiting Apple Approval
- **Stage**: TestFlight preparation complete
- **Last Activity**: January 2-3, 2026
- **Blocker**: Apple Developer Account approval pending

## Tech Stack
- **Mobile App**: React Native + Expo
- **Admin Dashboard**: React + Vite + TypeScript
- **Backend**: Supabase (PostgreSQL, Edge Functions)
- **Payments**: Clover API
- **Authentication**: OTP-based phone authentication

## Key Features
- Customer mobile app for booking dog park time slots
- Admin dashboard for park management
- Payment processing via Clover
- Real-time availability tracking
